# Active Work Queues

This folder owns active implementation queues and execution sequencing.

Use it for:

- outstanding work
- milestone sequencing
- acceptance/non-goal definitions
- active progress state

Do not use it for:

- durable architecture truth
- historical deep-dive writeups
- public product overview

Ownership split:

- `docs/todo/implementation.md`: primary active queue for current milestone.
- `docs/todo/ENGINEER_ENTRYPOINT.md`: dual-mode engineer contract and reporting format.
- `docs/Vision.md` and design docs: product/design authority.
